Firefly Wellness marketing site — a [Next.js](https://nextjs.org) project bootstrapped with [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app), ready to deploy on [Vercel](https://vercel.com).

## Getting Started

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

Edit the homepage in `src/app/page.tsx` and the root layout (e.g. metadata, global styles) in `src/app/layout.tsx`. Add more routes under `src/app/` (e.g. `src/app/about/page.tsx`, `src/app/contact/page.tsx`).

This project uses [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) to automatically optimize and load [Geist](https://vercel.com/font), a new font family for Vercel.

## Deploy on Vercel

1. Push this repo to a remote (GitHub, GitLab, or Bitbucket).
2. Go to [vercel.com](https://vercel.com) → **Add New Project** and import the `fireflywellness` repo.
3. Vercel will detect Next.js and set the build command and output automatically. Click **Deploy**.
4. Optional: add a custom domain under the project’s **Settings → Domains**.

No `vercel.json` is required; the connection is between your Git repo and Vercel’s dashboard. See [Next.js deployment docs](https://nextjs.org/docs/app/building-your-application/deploying) for more.
