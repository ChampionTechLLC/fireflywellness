# Social media icons

Add your icon assets here (e.g. SVG or PNG). Reference them in `src/data/social.ts` as `icon: "/social/filename.svg"`.

- Files are served at `/social/filename.ext`
- Suggested size: 24×24px or 48×48px for retina
- Use placeholder.svg as a fallback or copy it to instagram.svg, facebook.svg, etc. until you have final assets
